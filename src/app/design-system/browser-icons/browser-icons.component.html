Guide for naming of products and browser icons

Browser Icons

The (favicon) icon that appear in the browser tab

Please use the Transform mark logo

There is a selection of favicon sizes available which you can download.

For the main favicon itself, it's best for cross-browser compatibility not to use any HTML. Just name the file favicon.ico (included in the zip file) and place it in the root of your domain.

There are further options which might need to be implemented depending on the application. See this useful favicon cheat sheet
