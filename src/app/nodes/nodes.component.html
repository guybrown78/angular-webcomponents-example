Nodes Node - designed to be used within a card A tf-node is a flexbox row. It will take the full width of any parent that allows flexbox to flow, eg tf-main-content, tf-card and tf-node-list (see Node List card below). Nodes are naturally a row, therefore items inside them will flow along the 'x' axis. However, it can be used to flow as a column by adding a singular child component inside set to column.
